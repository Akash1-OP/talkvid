# 🎥 TalkVid - Real-Time Video Calling & Social Platform

TalkVid is a full-stack real-time communication platform that enables users to connect through video calls, instant messaging, and social networking features. Built using modern web technologies, TalkVid delivers a seamless and interactive communication experience.

---

## 🌐 Live Demo

https://talkvid.onrender.com

> ⚠️ The application is hosted on Render's free tier. The backend may take a few moments to wake up on the first request.
